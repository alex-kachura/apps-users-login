## Prerequisites

You'll need to have [NodeJS with NPM](https://nodejs.org/en/) installed

## Install

Install dependencies:
```sh
npm i
```

alternatively, if you have yarn
```sh
yarn
```

## Run

Run frontend:

```sh
npm start
```

alternatively, if you have yarn

```sh
yarn start
```

The app will be available at [http://localhost:3000](http://localhost:3000)

## Test

```sh
npm test
```

alternatively, if you have yarn

```sh
yarn test
```
